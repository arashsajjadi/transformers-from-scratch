# Tiny Datasets

These small datasets are included in the repository for fast, offline training.

## tiny_sentiment.csv

- 90 short English sentences
- Labels: positive, negative, neutral (30 each)
- Used in notebooks 05 and 06

## tiny_translation_en_fr.csv

- 100 simple English-to-French sentence pairs
- Uses only common words and ASCII characters
- Used in notebook 07
