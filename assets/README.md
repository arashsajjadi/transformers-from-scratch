# Assets Directory

This directory contains static assets for the course documentation.

## Structure

```
assets/
├── diagrams/        <- Architecture diagrams (text-art or image files)
├── figures/         <- Reference figures used in documentation
└── sample_outputs/  <- Example output images from the course
```

## Sample outputs

The `sample_outputs/` directory contains example PNG files showing what
successful outputs look like. These are committed to the repository for
reference. They are NOT gitignored.
